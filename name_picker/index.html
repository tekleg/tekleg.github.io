Random Name Picker App

Random name picker is an online tool to quickly picks one random name or winner from the list of the names.

Provide your list of names in the text field area. Split each name by commas, except the very end and click the starting button to get the winner name.

To keep playing, copy the listed names from the text field area. Refresh the link and past the names and continue playing.


Click start to pick the random name!
